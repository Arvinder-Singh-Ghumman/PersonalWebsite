Html cheatsheet

I will share my html cheatsheet that helps me alot.

TAG Usage
<HTML> Root element
<head> head of the document where contents like title tag go and other useful tags mentioned below
<title> Title of the page
<body> Body of the document
<h1> to <h6> Heading tags
<p> Paragraph
<a> hyperlink
<ul/ol> unordered/ ordered lists respectively
<li> List items, used in ul or ol tags
<img> image tag
<table> Used to make a table
<tr> Row of table
<th/td> Table heading or Table data respectively
<div> Used to create divisions
<span> used to contain certain text to give them specific properties through css
<!-- --> Used to write comments
