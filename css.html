CSS cheatsheet

These are the most common selectors and properties I usually need while working with css, i have put them in two seperate tables

Selector Description
* Universal Selector (selects all elements)
#id selects a specific element by ID
.class To select elemnt with certain class
selector1, selector2 Comma used to use multiple selectors at once
> selects direct child elements
:hover Hover Pseudo-class
:nth-child(n) nth-child Pseudo-class
:nth-of-type(n) nth-of-type Pseudo-class
[attribute] selects by attribute
[attribute=value] Attribute having a certain value is selected
:first-of-type Selects first of a certain type
:last-of-type Selects last of a certain type
Property Description
font-family Changes the font family of certain words, sentences, paragraphs, etc.
font-style Changes text: normal, oblique, and italics.
font-weight Used to specify the weight of the font.
color Sets the text color.
margin Sets the margin space outside the border of an element.
padding Sets the padding space inside the border of an element.
background Sets various background properties, including color and image.
